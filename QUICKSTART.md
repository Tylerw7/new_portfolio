## Get Your Portfolio Running in 3 Steps

### Step 1: Install Dependencies
```bash
cd portfolio
npm install
```

### Step 2: Run Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Visit: http://localhost:3000

## Next Steps

### 1. Customize Your Content
- Edit `components/Hero.tsx` - Update your name and title
- Edit `components/About.tsx` - Add your bio and skills
- Edit `components/Projects.tsx` - Add your projects
- Edit `components/Experience.tsx` - Add your work history
- Edit `components/Contact.tsx` - Update contact info

### 2. Customize Colors (Optional)
Open `app/globals.css` and modify the CSS variables:
```css
:root {
  --primary: 99 102 241;      /* Your main color */
  --secondary: 139 92 246;    /* Your secondary color */
  --accent: 236 72 153;       /* Your accent color */
}
```

### 3. Deploy to Vercel
1. Push to GitHub
2. Go to vercel.com
3. Import your repository
4. Click Deploy!
